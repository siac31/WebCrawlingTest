Welche Möglichkeiten stehen Ihnen mit dem CBR-System zur Verfügung?

Betätigen Sie den Button Diagnose starten, wenn Sie umgehend mit der dialogbasierten Störungsdiagnose beginnen möchten. Sie haben alternativ die Möglichkeit, über das Menü eigenständig auf die Informationen der Anlage, sowie bekannte Fälle zuzugreifen. Wenn Sie auf die dialogbasierte Analyse verzichten möchten, können sie über den Menüpunkt Fragkatalog eigenständig Fragen auswählen, die Sie beantworten möchten (nicht empfohlen).
